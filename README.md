# React 19 Playground

An interactive demo app exploring the new features and hooks introduced in React 19, built with TypeScript and Vite.

## Demos

| #  | Feature              |
|----|----------------------|
| 01 | useActionState       |
| 02 | useFormStatus        |
| 03 | useOptimistic        |
| 04 | use() hook           |
| 05 | ref as prop          |
| 06 | ref cleanup          |
| 07 | Context as Provider  |
| 08 |`useDeferredValue     |
| 09 | Document Metadata    |
| 10 | Asset Loading        |
| 11 | Error Handling       |
| 12 |  startTransition     |
| 13 | Custom Elements      |

## Tech Stack

- React 19.2.0 + TypeScript
- Vite 7.3.1
- React Router DOM 7.13.1

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command             | Description              |
|---------------------|--------------------------|
| `npm run dev`       | Start Vite dev server    |
| `npm run build`     | Production build         |
| `npm run preview`   | Preview production build |
| `npm run lint`      | Run ESLint               |
